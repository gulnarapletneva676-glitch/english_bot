# 🚀 Деплой English Trainer Bot на PythonAnywhere

## 1. Регистрация и вход

1. Зайди на https://www.pythonanywhere.com
2. Зарегистрируйся (бесплатный аккаунт Beginner подойдёт)
3. Войди в аккаунт

---

## 2. Открытие консоли

На дашборде нажми **Consoles** → **Bash** → **New console**

---

## 3. Загрузка файлов

### Вариант A — через Git (рекомендуется)

Если проект на GitHub:
```bash
git clone https://github.com/ВАШ_ЛОГИН/english_bot.git
cd english_bot
```

### Вариант B — ручная загрузка

1. На дашборде нажми **Files**
2. Создай папку `english_bot`
3. Загрузи все файлы по одному через кнопку **Upload a file**

---

## 4. Установка зависимостей

В Bash консоли:
```bash
cd ~/english_bot
pip3.10 install --user aiogram==3.7.0 aiosqlite==0.20.0
```

> ⚠️ На PythonAnywhere используй `pip3.10` (или ту версию Python, что выбрал)

---

## 5. Настройка переменных окружения

Открой файл `config.py` и замени значения:
```bash
nano ~/english_bot/config.py
```

Измени строки:
```python
BOT_TOKEN = "ВАШ_ТОКЕН_ОТ_BOTFATHER"
ADMIN_IDS = [ВАШ_TELEGRAM_ID]  # Узнать свой ID: @userinfobot
```

Сохрани: `Ctrl+X` → `Y` → `Enter`

---

## 6. Тестовый запуск

```bash
cd ~/english_bot
python3.10 bot.py
```

Если видишь `Bot started!` — всё работает. Остановить: `Ctrl+C`

---

## 7. Настройка Always-On Task (чтобы бот работал 24/7)

> ⚠️ Always-on tasks доступны только на платных аккаунтах (Hacker от $5/мес)
> На бесплатном аккаунте бот работает только пока открыта консоль

### На платном аккаунте:

1. Перейди на вкладку **Tasks**
2. Нажми **Add a new scheduled task**
3. Выбери **Always-on**
4. В поле команды введи:
   ```
   /home/ВАШ_ЛОГИН/english_bot/run.sh
   ```
5. Создай файл `run.sh`:
```bash
cat > ~/english_bot/run.sh << 'EOF'
#!/bin/bash
cd /home/ВАШ_ЛОГИН/english_bot
python3.10 bot.py
EOF
chmod +x ~/english_bot/run.sh
```

### На бесплатном аккаунте (workaround):

Используй Scheduled Task каждые 24 часа с перезапуском, или запускай вручную через консоль.

Альтернатива — задеплой на **Railway.app** (бесплатно, проще для Telegram-ботов):
```
https://railway.app
```

## 8. Проверка работы бота

1. Открой Telegram
2. Найди своего бота по имени
3. Напиши `/start`
4. Должно появиться приветственное сообщение с меню

## 9. Структура файлов

```
english_bot/
├── bot.py           # Точка входа
├── config.py        # Настройки (токен, admin IDs)
├── database.py      # SQLite через aiosqlite
├── handlers.py      # Все обработчики команд
├── keyboards.py     # Клавиатуры
├── scheduler.py     # Ежедневные напоминания
├── requirements.txt # Зависимости
└── english_bot.db   # База данных (создаётся автоматически)
```

---

## 10. Команды бота

| Команда | Описание |
|---|---|
| `/start` | Запуск / регистрация |
| `📚 Начать урок` | Новый урок (5 слов + тест) |
| `👤 Профиль` | Профиль с прогрессом |
| `📊 Статистика` | Твоя статистика |
| `ℹ️ Помощь` | Справка |

### Команды администратора

| Команда | Описание |
|---|---|
| `🔧 Админ-панель` | Открыть панель управления |
| `/ban <id>` | Заблокировать пользователя |
| `/unban <id>` | Разблокировать пользователя |
| `/addword` | Добавить новое слово |
| `/delword <id>` | Удалить слово по ID |

---

## 11. Получение Telegram ID

Напиши боту @userinfobot — он пришлёт твой ID. Вставь его в `ADMIN_IDS` в `config.py`.

---

## 12. Получение токена бота

1. Найди в Telegram @BotFather
2. Напиши `/newbot`
3. Следуй инструкциям
4. Скопируй токен и вставь в `config.py`
